# Асинхронный JavaScript

**Асинхронность** — это способ выполнения кода, при котором некоторые операции не блокируют главный поток. Особенно важно для запросов к серверу, таймеров и работы с файлами.

## Почему это важно?

JavaScript — однопоточный язык. Если бы все операции выполнялись синхронно, загрузка большого объёма данных «заморозила» бы страницу до завершения загрузки.

## Колбэки (Callbacks)

Самый старый способ работы с асинхронным кодом:

```javascript
setTimeout(() => {
	console.log("Прошла 1 секунда");
}, 1000);

function loadData(callback) {
	// имитация загрузки
	setTimeout(() => {
		callback({ name: "Данные" });
	}, 500);
}
```
