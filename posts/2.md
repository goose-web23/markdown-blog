# Работа с DOM

**DOM (Document Object Model)** — это программный интерфейс, который представляет HTML-документ в виде дерева узлов. JavaScript позволяет изменять это дерево в реальном времени.

## Что такое DOM?

DOM — это мост между HTML и JavaScript. Браузер загружает страницу, парсит HTML и создаёт DOM-дерево, с которым мы можем работать.

```javascript
// Доступ к элементам
const header = document.getElementById("main-header");
const buttons = document.querySelectorAll(".btn");
```
